Requirements:

● Se doreşte realizarea unui instrument Web (i.e. o extensie de navigator Web) care permite indicarea schimbărilor ce au survenit în cadrul unei pagini Web faţă de vizita precedentă. Se vor oferi posibilităţi de configurare (e.g., lista URL-urilor care nu vor fi luate în consideraţie, modul de redare a zonelor actualizate ale documentului, întreruperea temporară a funcţionalităţii instrumentului etc.), cu suport pentru export/import al datelor de configurare la nivel local sau recurgând la un serviciu de stocare în "nori" precum Dropbox, OneDrive etc.

How to install aplication:

1)Firs of all you need to download the aplication from git. By click on the "Download zip" like in image

2)Go where you downloaded the archive and unzip it.

3)Now, you need to open Google Chrome,then in the upper right corner click on the "Customize and control Google Chrome".Go to the "More tools" and click on "Extension"

4)You need to check "Developer mode", then you will see three buttons. Click on the first one, "Load unpacked extension..."

5)Go to the folder where you unzip the file you have download from git, and select the folder named "extension"
