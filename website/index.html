Assuming you’ve successfully associated the appropriate Protege libraries with your Eclipse, lets get started with some simple HOW-TOs in programmatical Protege ontology manipulation; For this tutorial I will be using the RadLex Lexicon/Dictionary. The first thing you will need is the actual project files. Mine are called : RadLex.pins, RadLex.pont, and RadLex.pprj These files are created when you save a Lexicon/Dictionary in Frames format using Protege. Create a pointer to them in your java code. I have mine in in C:\smelo\RadLex.pprj You only have to point to the *.pprj file.
